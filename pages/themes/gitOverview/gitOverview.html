Initialize a Git repository


				$ git init
git will start to track the Current Working Directory (CWD).
A hidden .git folder is created.

Stage files/changes

Also known as adding files content to the index


				# to stage only certain file/path:
				$ git add path

				# to stage all files in CWD:
				git add -A
The staging are contains all the changes that will be added to the local repository after the commit
You can remove (unstage) files from the index, with: 

					git rm file

Commit files to local repository


				# commit files - the default editor for git will be opened to write your Commit Message
				$ git commit

				# commit and write the Commit Message
				$ git commit -m'Your Commit Message'
You can set a default editor by your choice:
How can I make Sublime Text the default editor for Git?

Open a Terminal/Command Prompt


  			# Create your project directory
  			$ mkdir my_project

  			# go into it
  			$ cd my_project

  			# initialize the new repository
  			$ git init

  			# do some work and save your files
  			$ echo "I'm working hard..." > file1.txt

  			# stage all your work
  			$ git add -A

  			# commit your work
  			$ git commit -am'Work 1 Done'
