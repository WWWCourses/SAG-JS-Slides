What is SPA

SPASinglePageApplication

SPA is a web application or web site that load a single HTML page and dynamically update only needed part of the page as the user interacts with the app.

SPAs use JavaScript, AJAX and HTML5 to create fluid and responsive Web apps, without constant page reloads.
