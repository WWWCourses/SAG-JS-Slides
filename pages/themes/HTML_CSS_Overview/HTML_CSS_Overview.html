HTML syntax

HTML syntax

An HTML document consists of HTML elements!
Each element has:
<opening tag> content </closing tag>
The content can be text, or another HTML element
Some HTML elements are empty - they have no content, and according to HTML5 - no closing tag. For example:
<img src="images/cats.png">

			<h1>Hello World</h1>
			<h1>Hello <i>World</i></h1>
			<h1 class="primary">Hello World</h1>
HTML5 defines the tag names as case-insensitive, but the convention is lower-case to be used.

Main attributes

action
URL to the server-sede program which will handles the user input
method
Specifies the HTTP request method:
  • get - defines a GET request. Form data will be attached in the URL
  • post - defines a POST request. Form data will be attached in the request body


Never use GET method when the form will contain sensitive information, like password fields.
